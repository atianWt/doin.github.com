2015.9.9~2015.9.23

- **1.2活动主题**<br/>
主标题：9月大学生信用节
副标题：芝麻开门，点亮大学生活

- **1.3背景和目标**<br/>
1）学生人群mindshare <br/>

- **1.4目标人群** <br/>
3036所高校，约2500万左右在校学生 <br/>
本次不涉及中专类学校。 <br/>

###66信用日
**1.1 活动目的**<br>
    通过66信用日预热活动，为6月6日活动主会场炒热氛围，通过前期的活动，让用户提前感知信用日并且持续给主会场引流，让更多的用户在66当天回流到活动的主会场参与活动。<br>
**1.2 活动时间**<br>
2016.06.01 00:00——2016.06.06 24:00<br>
**1.3 活动目标**<br>
单商户单场景调用1000w次<br>
**1.4 活动对象**<br>
全体芝麻信用准入用户，每一个账号RoleID视为一位用户。<br>
